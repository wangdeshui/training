* HTML5 start-up
	
	```html
	<!DOCTYPE html>
	<html lang="en">
		<head>
			<meta charset="utf-8">
			<title>A Tiny HTML Document</title>
			<link href="styles.css" rel="stylesheet">
			<script src="scripts.js"></script>
		</head>
		<body>
			<p>Let's rock the browser, HTML5 style.</p>
		</body>
	</html>
	
	```
